\subsubsection{气体分子运动的微观模型}
\label{sec:org0c4e01a}
\begin{itemize}
\item 气体分子本身的体积很小可以忽略
\item 气体分子不断作规则运动均匀地分布在容器中
\item 分子彼此、分子与器璧的碰撞是完全弹性的
\end{itemize}
这种假设在低压下分子间距远大于分子本身的体积时是适用的
